两天实战：从构思到上线与推广 · 周五 20:00 腾讯会议集合

周五 20:00 · 开幕

腾讯会议：开场 & 方向讨论

· 主持人介绍两天流程 / 规则 / 目标
· 我自我介绍 & 发起初衷、社区愿景

· 确认个人方向 / 组队意向
周五 21:00 起 · 自由互动

群内交流 / 方向确认 / 继续打磨

· 组队沟通、想法完善
· 有经验同学主动答疑引导
周六 · 全天执行

项目推进：设计 / 开发 / 部署 / 推广 / 测试

· 设置中期 Check-in（中午 / 下午 / 晚上）
· 群里互帮互助，遇到问题及时抛出
· 记录问题与待办，保持小步快跑
群主临时外出

值班答疑机制

· 我可能阶段性离线，会提前安排助教值班答疑；若行程调整，会在群内同步
· 常见问题清单 / 教程文档提前发放，可自助查询
周日 · 白天推进

继续调试 / 部署 / 上线 / 推广

· 按节点对齐进度，补齐最小可用版本（MVP）

周日 19:00 · 收集

统一提交成果（模板）

· 项目名 / 简介 / 功能要点 / Demo 链接或截图
· 遇到的问题 / 下一步计划 / 希望的支持
周日 20:00 · 总结 & 表彰

腾讯会议：展示 / 分享 / Q&A

· 优秀项目展示与经验分享，开放问答
· 主持人总结与表彰，公布下一期预热信息
