三个关键卡点

  • 工具选择：不知道选谁，不敢下手。

  • 与 AI 对话：不会拆解，不知从何问起。

新手三步上线（现场演示）

以“静态站点”为最小可交付：

  1. 生成：让 AI 产出一页式站点（含 CTA 按钮）。
  2. 部署：用 Cloudflare Pages / Vercel / Netlify 任意其一。
  3. 收集：按钮直达麦客表单，承担信息与支付。

域名与后台功能放到第二阶段，先把“可见链接”跑起来。

来自现场的好建议

吴伟：聚焦“能上线”的路径，先跑通免费托管与表单再扩展。

孙攀：兴趣驱动 + 引导式提示词，先把自己的“小工具”跑起来。

新手入门包（更人话、一步步来）

先学会问，再做一页网页，最后把它放到网上。会问，就已经走了一半。

从提问开始

先学会提问（开口就能用）

把 AI 当作会帮忙的朋友，按这个顺序开口：

  1. 我想做的主题是「____」，请列 5 个具体场景（要接地气）。
  2. 选场景 A，写一段 300–500 字的销售文案：主标题/3 个卖点/一个行动按钮。
  3. 请把这段文案做成一页网页（HTML/CSS），按钮文案写“立即报名/试用”。
  4. 按钮点开后，需要收集：姓名、微信、需求，请设计表单字段。
  5. 润色成“价值-证据-行动”的逻辑，再压一遍字数。
做网页

做一个一页网站（复制就能改）

  • 把 AI 给的 HTML/CSS 复制到本地文件夹（如 index.html）。
  • 保留一个明显的大按钮，先直达报名/收集，不急着做登录、后台。
  • 能看到页面、能点按钮，就算第一步完成。
放到网上

放到网上（3 种任选）

  • Cloudflare Pages：新建项目 → 上传文件夹 → 复制链接分享。
  • Vercel：New Project → Deploy → Upload → 复制链接。
  • Netlify：Add new site → Deploy manually → 上传 → 复制链接。

域名与后台功能放第二阶段；先拿到“可见链接”，再收集 5 条反馈。
